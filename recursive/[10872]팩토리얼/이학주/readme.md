### 팩토리얼 풀이

처음에 팩토리얼 문제를 보고 for문이 생각났다.
for문으로 푼다면 간단하게 풀 수 있는 문제였지만 재귀적 사고방식을 적응하기위해서 재귀로 풀어보려고 재귀에 대해서 공부를 하였다.
처음에는 어렵고 어색했다. 그렇지만 계속해보니 재밌었다.

***

입력] N
변수 N은 1부터 N까지의 곱을 구하기 위해서 입력받는 값이다.
즉, N!을 구하기 위한 변수이다.

메인 함수에서는 입력과 출력만을 구현했고, factorial이라는 함수를 생성하여 계산을 하도록 해주었다.
계산식은 간단하다.

return (n * factorial(n-1));

입력받은 n과 n-1을 계속해서 곱해주는 로직이다.

ex) N = 4;

4 * factorial(n-1);
    3 * factorial(n-1);
        2 * factorial(n-1);
            1 * factorial(n-1);
                0 -> 0을 곱하면 0이 되기때문에 0일때 1로 리턴하는 조건문을 걸어주었다.
                1

출력] 24
