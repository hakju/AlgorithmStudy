# [동전 0(11047)](https://www.acmicpc.net/problem/11047)

처음에는 굉장히 어렵게 생각했었다.
그래서 재귀로 풀려고 시도하였지만 메모리초과라는 결과가 나왔다.

여러번 시도한 결과 안풀려서 다음 문제를 먼저 풀고왔다.
재귀를 포기하였다.
문제가 풀렸다.

이 문제는 그냥 쉽게 생각하면 된다.
몫, 나머지, 제수, 피제수로 구분지어서 생각하였다.

우선 입력받은 K가 피제수이다.
그리고 vector v에 저장되는 값들이 제수들이다.
vector v는 오름차순으로 정렬되어있다.
최솟값을 구하는 것이기 때문에 제수집합에서 큰 수부터 제수에 넣었다.

피제수 = K
제수 = v
몫 = temp
나머지 = K (다음 피제수가 되어야하기 때문이다.)
```c++
for (int i = N - 1; i >= 0; i--) {
        if (K >= v[i]) {
            temp = K / v[i];
		    count = count + temp;
		    K = K % v[i];
        }
    }
```
