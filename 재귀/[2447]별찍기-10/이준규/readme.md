3^2 > 3^1 로 재귀호출할때
큰범위의 ( i , j ) 칸번호 작은 범위의 칸 번호 (x, y)에 대한 상관관계로 별을 찍는다

(x+ 3^1*i , y+ 3^1*j) 이다.
